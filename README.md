#  CreditVision AI: Explainable Credit Risk System

This project is a high-level Machine Learning solution designed to predict financial credit risk using the German Credit Dataset. 

##  Key Innovations
- **XAI (Explainable AI):** Integration with **SHAP** to interpret model decisions, moving away from "black-box" predictions.
- **Robust Pipeline:** Full data preprocessing, feature engineering, and performance evaluation.
- **Fintech Ready:** Focus on minimizing False Negatives to reduce financial loss.

##  Tech Stack
- **Language:** Python 3.12
- **Core:** Scikit-Learn, Pandas, NumPy
- **Interpretability:** SHAP (Shapley Additive Explanations)
- **Visualization:** Seaborn, Matplotlib
